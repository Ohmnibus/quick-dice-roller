Rolagem de Dados Rápida

##Description:4000##
Rolagem de Dados Rápida (Quick Dice Roller) é um app feito para aficionados em RPG de mesa. Ele permite simular quase todas as complexas roladas necessárias para RPG e jogos de tabuleiro.

Os dados podem ser adicionados a vontade e organizados em sacolas de dados. Cada dado é definido por uma fórmula como  <i>2d6+5</i> bem como funcões especiais como <i>rak(1d6,4,3)</i> que permite simular os complexos sistemas de jogo. Um prático construtor de fórmulas facilita o processo e permite o acesso para uma ajuda on-line. Verifique a lista de todas as funções disponíveis aqui: http://goo.gl/mZkcQS

Graças as "variáveis nomeadas", bem como manter um registro dos resultados dos jogos, você pode definir fórmulas usando parâmetros variáveis. Você pode criar, por exemplo, a variável nomeada "Nível" com a etiqueta "nvl", é usar a fórmula <i>(nvl)d6</i> para determinar o dano da bola de fogo. Então, quando o seu mago subir de nível, você não precisa editar a fórmula: basta mudar o valor do "Nível"!

<b>O que você pode fazer com o Rolagem de Dados Rápida:</b>

★ Rolar os dados com um toque
★ Organizar dados em diferentes bolsas de dados
★ Use as mesmas fórmulas que você está familiarizado (<i>xdy+z</i>)
★ Mantenha o registro dos atributos e os use nas fórmulas
★ Leia ou escute o resultado da rolada de uma maneira clara
★ Verifique os benefícios das roladas feitas
★ Aplique os modificadores durante a rolada com um toque
★ Some a rolada anterior automaticamente ou quando quiser
★ Cole o resultado da última rolada na área de transferência em outro aplicativo
★ Mantenha o registro das últimas roladas feitas
