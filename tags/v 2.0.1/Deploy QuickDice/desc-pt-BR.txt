##Title:30##
Rolagem de Dados Rápida

##Description:4000##
Rolagem de Dados Rápida (Quick Dice Roller) é um app feito para aficionados em RPG de mesa. Ele permite emular quase todos as complexas roladas de dados usadas em jogos de RPG e tabuleiro.

Podem-se acrescentar dados a vontade, usando fórmulas simples como <i>2d6+5</i>, bem como, com funções especiais do tipo <i>rak(1d6,4,3)</i> que simulam sistemas de jogo mais complexos. Um prático construtor de fórmulas facilita o processo e permite o acesso para uma ajuda on-line.

Confira neste endereço a lista das funções do app: http//goo.gl/TTjb9

Não se preocupe! Você não precisará escrever a fórmula o tempo todo! Uma vez definida, o dado será rolado todas as vezes que você quiser com um simples toque no ícone correspondente.

"Variável Nomeada", bem como um registro de valores do jogo, permitem que você faça fórmulas baseadas nesses dados. Você pode criar, por exemplo, uma Variável Nomeada "Nível" com a etiqueta "nvl", então usá-la na fórmula "(nvl)d6" para determinar o dano da bola de fogo - desse jeito você não precisa editar a fórmula toda vez que seu mago subir de nível: basta somente mudar o valor da Variável Nomeada "Nível"!

Rolagem de Dados Rápida te permite criar bolsas de dados diferentes (para organizar os dados por sistema de jogo, personagem ou por qualquer razão que você queira) e é possível trocar de uma bolsa para a outra rapidamente.

O app permite configurar rolagens em cadeia, duas rolagens feitas rapidamente são consideradas em conjunto. Ou, se esse conjunto de rolada acontecer com frequência, sempre é possível criar um novo dado que some o resultado de duas rolagens separadas (isto é, <i>1d6+1d8</i>).

Gosta de jogar por sala de bate-papo ou por e-mail? Rolagem de Dados Rápida permite que automaticamente você copie o último resultado de sua rolagem para a área de transferência, então compartilhar passa a ser somente o comando <i>colar</i>.

Rolagem de Dados Rápida é um dos poucos apps da categoria (se não for o único) que permite aplicar modificadores após a rolagem ter sido feita, facilmente com um simples toque. O guerreiro está cego? Sem problema: toque para a rolagem do ataque e toque para aplicar o modificador de -4.
